# GPS-route-tracker

A comprehensive Django-based live GPS route tracker with real-time road snapping (powered by OpenRouteService), offline IndexedDB sync, and a premium glassmorphism user interface. 

Features include:
- Real-time location tracking with Motion Filter smoothing
- Real-time road geometry snapping
- Offline route storage and automatic online syncing
- Date-wise Route History playback on Leaflet Maps
- User Authentication (Login/Register)
